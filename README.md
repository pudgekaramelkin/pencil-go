# Pencil GO  
Сетевая игра в реальном времени, где один рисует, а остальные угадывают. Сделана на React, Socket.IO и Konva.js.

## Особенности
- 3‑10 игроков в комнате  
- Рисование в реальном времени с несколькими цветами и толщинами кисти  
- Поочерёдный игровой процесс: ролевая смена художника  
- Постепенное раскрытие слова (по 1 букве каждые 5 секунд)  
- Система очков с бонусами за скорость  
- Управление хостом (выгон игроков, настройка параметров)  
- UI, адаптированный под мобильные устройства  

## Технологический стек  
**Frontend:** React + Vite + TypeScript + Tailwind CSS + Konva.js  
**Backend:** Node.js + Express + Socket.IO  
**Состояние:** Zustand  

## Установка и запуск  

### Разработка
1. Установите зависимости:  
   ```bash
   npm install
   ```
2. Запустите клиент и сервер одновременно:  
   ```bash
   npm run dev
   ```
   - Клиент: <http://localhost:3000>  
   - Сервер: <http://localhost:3001>  

### Продакшен
Соберите образ и запустите через Docker:  
```bash
docker-compose up --build
```
Доступно по адресу <http://localhost:3001>  

## Правила игры
1. Создайте комнату или присоединитесь к существующей (код из 6 символов).  
2. Хост задаёт настройки и стартует игру, когда все готовы.  
3. Каждый раунд один игрок рисует, остальные пытаются угадать слово.  
4. За правильный ответ начисляются 100 очков + бонус за оставшееся время.  
5. Художник получает 50 очков за каждого угадавшего.  
6. Слово раскрывается по 1 букве каждые 5 секунд.  
7. Раунд заканчивается, когда время истекает или все угадают.  
8. Побеждает игрок с наибольшим количеством очков после всех раундов.
